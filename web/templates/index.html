Онлайн трейд


  • Счёт
  • УПД

Для этого клиента необходимо в файлы с счетом добавлять данные о ГТД, штрихкоде, стране производства

Порядок действий
1. Выгружаем из 1с файл xlsx с заказом покупателя
2. Выбираем файл с помощью кнопки Файл и нажимаем Пуск и ждём когда странится обновится
3. После выполнения скрипта, в этом же файле напротив каждой позиции добавлены нужные данные

Скрипт помогает создать xml файл с электронной версией УПД для отправки в Онлайн-Трейд на основе счета Эксель. Данные о товарах содержатся в справочнике directory.xlsx. В файле template_main3.xml содержится болванка на основе которой будет создан новый файл.
