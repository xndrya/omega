Вайлдберриз | Создать УПД в xml формате

Скрипт помогает создать xml файл с электронной версией УПД для отправки в Вайлдберриз на основе счета Эксель. Данные о товарах содержатся в справочнике directory.xlsx. В файле template_main.xml содержится болванка на основе которой будет создан новый файл.
  1. Выгружаем из 1с файл xlsx с заказом покупателя
  2. Выбираем файл с помощью кнопки Файл и нажимаем Пуск



  3. Ждём дальнейших инструкций

Вайлдберриз | Заполнить файл дефицита, остатков наименованиями позиций

Скрипт нужен для записи наименований позиций в файл с остатками товаров в Вайлдберриз. В исходном содержатся артикулы, штрихкоды, а наименований нет. Это усложняет задачу анализа и планирования ассортимента.
  1. Выгружаем из 1с файл xlsx с заказом покупателя
  2. Выбираем тип файла в который требуется добавить информацию переключателем
  3. Выбираем файл с помощью кнопки Файл и нажимаем Пуск
  4. Ждём дальнейших инструкций
